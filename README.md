# Meditation Helper | 명상 헬퍼

숨 쉬기 호흡을 시각화하여 명상을 돕는 웹 앱입니다.

## 기능

- **호흡 시각화**: 들숨 시 공이 위로, 날숨 시 아래로 움직여 호흡 리듬을 따라갈 수 있습니다.
- **카테고리**: 휴식, 집중, 수면, 스트레스 해소, 아침 명상, 에너지, 평온 등 다양한 호흡 패턴을 선택할 수 있습니다.
- **다국어**: 한국어, English, 日本語, Español 지원.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 표시되는 주소(예: http://localhost:5173)로 접속하세요.

## 빌드

```bash
npm run build
npm run preview   # 빌드 결과 미리보기
```

## 기술 스택

- React 18 + Vite 5
- CSS (그라디언트, 트랜지션)
